# AgenTEC
## Descrição
O AgenTEC é um sistema de gerenciamento de laboratórios projetado para otimizar a reserva de espaços e a organização de materiais. A plataforma permite que usuários agendem horários nos laboratórios de forma eficiente e solicitem "kits" personalizados, que incluem vidrarias e reagentes necessários para seus experimentos.

**_Funcionalidades_** 
Agendamento de Laboratórios: Sistema de calendário para horários disponíveis e realização de reservas.

Criação de Kits: Funcionalidade para montar kits com vidrarias e reagentes específicos para cada experimento.

Gerenciamento de Inventário: Controle de estoque de materiais de laboratório (vidrarias e reagentes).

Autenticação de Usuários: Sistema de login e perfis de usuário para professores e técnicos.

Interface Intuitiva: Design focado na experiência do usuário para facilitar a navegação e o uso do sistema.

**_Tecnologias Utilizadas:_**
- HTML
- CSS
- JavaScript
